# JurnalKu 📒
A modular Android app built with Kotlin + Jetpack Compose  
Features: Finance tracking, journaling, export to PDF

## 📦 Modules
- app/
- core/core-ui
- core/core-data
- core/core-domain
- feature/auth
- feature/journal
- feature/finance
- feature/export

## 🛠️ Built With
- Kotlin
- Jetpack Compose
- MVVM + Clean Architecture
- Koin
- Room
